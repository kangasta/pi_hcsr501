## Testing

Run unit tests with command:

```bash
cd pi_hcsr501

python3 -m unittest discover -s tst/
```

Get test coverage with commands:
```bash
cd pi_hcsr501

coverage run -m unittest discover -s tst/
coverage report -m
```
